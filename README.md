# MyBank Analytics Project

## 📊 Overview
This project explores loan data to generate insights into customer behavior, payment trends, and risk factors.  
It demonstrates analytics using **SQL, Excel, Power BI, and Tableau**.

## 🚀 Features
- Interactive dashboards with KPIs
- SQL scripts for cleaning and transformations
- Power BI and Tableau packaged workbooks
- Sample datasets for demonstration

## 📂 Project Structure

/data        → sample datasets (CSV/Excel, anonymized)
/scripts     → SQL queries, cleaning scripts
/dashboards  → Power BI (.pbix), Tableau (.twbx)
/docs        → PPTs, notes, screenshots


## 📖 Analysis Guide
The project covers the following tasks:

1. **Year‑wise Loan Amount Stats**  
   Trend analysis of loan amounts across different years.

2. **Grade and Sub‑Grade wise Revolving Balance (revol_bal)**  
   Distribution of revolving balances segmented by loan grade.

3. **Total Payment for Verified vs Non‑Verified Status**  
   Comparison of repayment behavior between verified and non‑verified borrowers.

4. **State‑wise and Last Credit Pull Date wise Loan Status**  
   Loan performance segmented by borrower’s state and credit pull timeline.

5. **Home Ownership vs Last Payment Date Stats**  
   Correlation between home ownership type and repayment timelines.
